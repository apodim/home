About GraniteOS

GraniteOS is the currently most-worked project at Apodim's Software. It's an open-source operating system simulator built on GDevelop, designed to provide a cutting-edge virtual computing environment with professional-grade features and open-source transparency.

Key Features

🔓 Open Source

Fully open-source codebase available for community contributions and transparency.

🖥️ Cross-Platform

Runs seamlessly on Windows, Linux, and macOS.

🎮 GDevelop-Based

Built using GDevelop game engine for optimal performance and compatibility.

⚙️ Professional OS Simulator

Realistic operating system simulation with modern UI/UX design.

👥 Community-Driven

Designed with community input and open collaboration.
